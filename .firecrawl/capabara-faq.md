## Frequently asked questions

### General Questions

[How do I get started with Capabara?](https://www.capabara.com/faq/#accordion-69c4c583272eb)

To get started, you can sign up for a Capabara account and choose the appropriate pack based on your needs. Detailed user guides are available for further assistance.

[What pricing plans are available?](https://www.capabara.com/faq/#accordion-69c4c58328c58)

Capabara offers various pricing plans, including the Starter Pack, Creator Pack, and Enterprise Pack, each tailored to different organisational needs.

Detailed user guides for CAI and CKS can be accessed by clicking on the Help Page button at the top right of the chat window.

### Introduction to Capabara Platform

[What is Capabara?](https://www.capabara.com/faq/#accordion-69c4c5832ad6c)

The Capabara Platform is a generative AI capability-as-a-service platform designed to enhance business productivity and learning. It provides organisations with tools to develop, govern, and manage digital transformation objectives using AI technology.

### Capabara AI Assistant (CAI)

[What is the Capabara AI Assistant?](https://www.capabara.com/faq/#accordion-69c4c5832c4e0)

The Capabara AI Assistant (CAI) is an interactive chat agent that allows users to access multiple popular language models such as OpenAI GPT, Claude, Gemini, Llama, and Perplexity. It includes over 50 AI tools and chatbots to enhance productivity and learning.

[How does CAI enhance productivity and learning?](https://www.capabara.com/faq/#accordion-69c4c5832ce12)

CAI offers a range of bundled tools across various areas like content writing, digital marketing, and project management. These tools help automate tasks, generate content, and provide insights, thereby boosting productivity and facilitating learning.

[What language models does CAI support?](https://www.capabara.com/faq/#accordion-69c4c5832d772)

CAI supports multiple language models, including OpenAI GPT, Claude, Gemini, Llama, and Perplexity, providing a safe environment for model switching within a conversation.

[How can I create custom AI tools with CAI?](https://www.capabara.com/faq/#accordion-69c4c5832e0bb)

Using the Apps builder, users can create custom AI tools and chatbots without coding knowledge. This feature allows for the design of role plays and the deployment of AI-enabled workflows.

[Is my data secure with CAI?](https://www.capabara.com/faq/#accordion-69c4c5832ea00)

Yes, CAI ensures data privacy and security by processing minimal personal information and using encryption methods like AES-256 for data storage.

### Capabara Knowledge System (CKS)

[What is the Capabara Knowledge System?](https://www.capabara.com/faq/#accordion-69c4c5833014a)

The Capabara Knowledge System (CKS) is a platform that enables organisations to create, deploy, and govern AI Knowledge Assistants using proprietary data. It leverages Retrieval Augmented Generation (RAG) technology for accurate responses.

[How does CKS utilise proprietary data?](https://www.capabara.com/faq/#accordion-69c4c58330ac0)

CKS uses Retrieval Augmented Generation (RAG) to combine information retrieval with generative models, ensuring responses are accurate and relevant to the proprietary data provided.

[What are the key features of CKS?](https://www.capabara.com/faq/#accordion-69c4c5833140a)

Key features include AI Knowledge Systems for creating custom chatbots, a File Manager for secure data handling, and a Governance Dashboard for monitoring performance and compliance.

[How is data security ensured in CKS?](https://www.capabara.com/faq/#accordion-69c4c58331d45)

CKS employs robust security measures, including AES-256 encryption for data at rest and secure access controls, to ensure data privacy and integrity.

### Capabara Capability Management (CCM)

[What is Capabara Capability Management?](https://www.capabara.com/faq/#accordion-69c4c58333649)

Capabara Capability Management (CCM) is a system designed to manage digital transformation objectives, turning AI recommendations into actionable tasks and managing performance and risk.

[How does CCM support digital transformation?](https://www.capabara.com/faq/#accordion-69c4c58333fb4)

CCM provides AI recommendations that can be transformed into tasks, along with capability templates for documents and events, aiding in the management of digital transformation efforts.

[When will CCM be available?](https://www.capabara.com/faq/#accordion-69c4c5833494d)

Capabara Capability Management is set to be launched by Q1 2025.

### Security and Privacy

[How does Capabara ensure data privacy?](https://www.capabara.com/faq/#accordion-69c4c583360c7)

Capabara adheres to strict privacy policies, processing minimal personal information and ensuring data is not used to train larger models.

[What security measures are in place?](https://www.capabara.com/faq/#accordion-69c4c58336a24)

Capabara employs robust security measures, including encryption, access control, and compliance with international standards, to protect user data.
